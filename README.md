# NexLearn - Advanced Learning Management System

NexLearn is a premium, full-stack Learning Management System (LMS) built with the MERN stack. It features a modern, responsive UI with GSAP animations and a professional folder structure following industry best practices.

## 🚀 Key Features
- **Premium UI**: Glassmorphism design with smooth GSAP animations.
- **Authentication**: JWT-based secure login and registration.
- **Modular Architecture**: Separate services, routes, and components for scalability.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **API Integration**: Centralized Axios configuration with request interceptors.

## 🛠️ Technologies Used
- **Frontend**: React.js (v19), GSAP (Animations), Axios, React Router Dom, Bootstrap.
- **Backend**: Node.js, Express, MongoDB (MERN Stack).
- **Styling**: Vanilla CSS with modern Glassmorphism.

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone <your-repo-link>
   cd lms-frontend
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up environment variables**:
   Create a `.env` file in the root directory and add:
   ```env
   REACT_APP_API_URL=http://localhost:5000/api
   ```

4. **Start the development server**:
   ```bash
   npm start
   ```

## 📂 Project Structure
```text
src/
├── assets/       # Media files and icons
├── components/   # Reusable UI components
├── context/      # Global state management
├── pages/        # Page views (Home, Login, Register)
├── routes/       # Route handling
├── services/     # API services (Axios instance)
└── styles/       # Global and component-specific styles
```
